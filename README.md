# gpxlatest

This tool helps you to find the latest timestamp of a gpx file.

## Install 

```sh
npm i -g gpxlatest
```

## Usage

```sh
Usage: gpxlatest [options]

This tool helps you to find the latest timestamp of a gpx file.

Options:
  -V, --version       output the version number
  -i ,--input <file>  specify the input file
  -h, --help          output usage information
```
